
# EduStream Launchpad

## Descrição
O EduStream Launchpad é uma plataforma educacional demonstrativa com portal do aluno, dashboard do professor, gerenciamento de cursos, avaliações e chat em tempo real. Desenvolvido em React + Vite.

---

## Instalação

```bash
git clone <repo>
cd edustream-launchpad
npm install
npm run dev
```

Acesse em:  
`http://localhost:5173`

---

## Navegação do Sistema

- `/` – Home  
- `/portal-aluno` – Portal do Aluno  
- `/dashboard-professor` – Dashboard do Professor  
- `/gerenciar-cursos` – Gerenciar Cursos  
- `/avaliacoes` – Avaliações  
- `/chat` – Chat ao Vivo
